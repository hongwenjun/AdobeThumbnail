### 安装方法
- 1.解压软件，先把GuiAdobeThumbnail放电脑适当目录
- 2.GuiAdobeThumbnail.exe  发送桌面方式，锁定到任务栏
- 3.需要先安装Everything软件，启用后台运行
- 4.开启 GuiAdobeThumbnail 输入 相关名  关键字搜索
- 5.文件名上双击打开文件，右键打开文件所在目录
